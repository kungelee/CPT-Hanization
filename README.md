## 文件结构及说明如下：
~~~ 
│  .gitignore
│  README.md   
│  translation.e  // 汉化编辑器源码（易语言）。
│  汉化编辑器.exe  // 汉化编辑器。
│  精易模块.ec  // 汉化编辑器调用模块。
├─doc
│  └─img
│          00.png
│
├─release  // 已编译的汉化文件，复制进CPT的language目录中即可直接使用
│      chinese_lsy9202_20190110.ptl
│
└─src  // 模板文件
        chinese_lsy9202.ts

~~~

## 汉化编辑器说明：  
编辑器使用易语言编写，杀毒软件可能会报毒，可以下载.e源文件自己编译运行。  
编辑器带调用有道词典自动翻译功能。
